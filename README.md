# TMS-frontend

Task manager system powered by [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
npm run dev
```

## Deploying the app

```bash
# create optimized build
npm run build
# serve
npm run start
```
